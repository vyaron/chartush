## Features

- Create various chart types (Bars, Circles, Donut, Rectangles)
- Animate charts with customizable settings
- Export animations as GIF
- Save charts to gallery
- Dark/Light theme support

## Tech Stack

- React + TypeScript
- Vite
- D3.js for charting
- gif.js for GIF export
